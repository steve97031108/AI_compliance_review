# AI 合规审查系统 — 使用说明

## 1. 项目简介

基于 **Spring Boot 3.2.5 + Vaadin 24.3.10** 的 AI 合规审查系统，集成 OpenAI GPT-3 兼容 API 实现内容合规自动审查。

## 2. 环境要求

| 项目 | 版本 |
|------|------|
| JDK | 21 |
| Maven | 3.6+ |
| 操作系统 | Windows / macOS / Linux |

## 3. 启动项目

### 方式一：命令行启动
```bash
cd C:\Users\HP\IdeaProjects\AI_compliance_review
mvn spring-boot:run
```

### 方式二：IntelliJ IDEA 启动
1. 打开 `src/main/java/com/scu/aicompliance/AiComplianceApplication.java`
2. 点击 `main` 方法左侧绿色三角 ▶，选择 **"运行"**
3. 或在右侧 Maven 面板 → **Plugins** → **spring-boot** → 双击 **spring-boot:run**

启动成功后控制台会显示：
```
Started AiComplianceApplication in X.XXX seconds
```

### 方式三：打包运行
```bash
mvn clean package -DskipTests
java -jar target/ai-compliance-system-0.0.1-SNAPSHOT.jar
```

## 4. 访问地址

启动后浏览器访问：**http://localhost:8080**

## 5. 登录账号

| 角色 | 用户名 | 密码 | 权限 |
|------|--------|------|------|
| 管理员 | `admin` | `admin123` | ADMIN — 全部功能 |
| 普通用户 | `user` | `user123` | USER — 基础功能 |

> 首次启动时系统会自动在 `src/main/resources/users.json` 生成默认账号。密码使用 BCrypt 加密存储。

## 6. 配置说明

### 系统配置文件
创建 `src/main/resources/config.json`（可选），内容示例：

```json
{
  "openaiApiKey": "sk-你的API密钥",
  "openaiBaseUrl": "https://api.openai.com",
  "modelName": "gpt-3.5-turbo",
  "temperature": 0.7,
  "maxTokens": 2048,
  "timeoutSeconds": 60,
  "enableOutputCheck": false
}
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `openaiApiKey` | OpenAI 兼容 API 密钥 | - |
| `openaiBaseUrl` | API 基础地址（支持任何兼容端点） | - |
| `modelName` | 模型名称 | - |
| `temperature` | 温度系数（0~1） | - |
| `maxTokens` | 最大生成 Token 数 | - |
| `timeoutSeconds` | 请求超时时间（秒） | - |
| `enableOutputCheck` | 是否启用输出合规检查 | `false` |

### 用户配置文件
用户数据存储在 `src/main/resources/users.json`，格式如下：

```json
[
  {
    "username": "admin",
    "passwordHash": "$2a$10$...",
    "role": "ADMIN"
  },
  {
    "username": "user",
    "passwordHash": "$2a$10$...",
    "role": "USER"
  }
]
```

删除该文件后重启，系统会重新生成默认账号。

## 7. 项目结构

```
src/main/java/com/scu/aicompliance/
├── AiComplianceApplication.java    # 启动入口
├── config/
│   └── SecurityConfig.java         # Spring Security 配置（BCrypt 密码编码器）
├── model/

└── util/
    └── JsonUtil.java               # JSON 读写工具类

src/main/resources/
├── users.json                      # 用户数据文件（自动生成）
└── application.properties          # Spring Boot 配置（可自行创建）
```

## 9. 常见问题

**Q: 启动报 "端口 8080 已被占用"**
在 `src/main/resources/application.properties` 中添加：
```properties
server.port=8081
```

**Q: IDEA 中报 "程序包 org.springframework 不存在"**
1. 右侧 Maven 面板 → 点击 🔄 **重新加载所有 Maven 项目**
2. 如仍有报错：**文件** → **清除缓存...** → **清除并重新启动**

**Q: 忘记密码怎么办**
删除 `src/main/resources/users.json` 后重启，系统会重新生成默认账号 `admin/admin123`。

**Q: 如何切换 JDK 版本**
修改 `pom.xml` 中 `<java.version>` 属性值，同时修改 `.idea/misc.xml` 中对应的 JDK 版本号，然后在 IDEA 中 **文件** → **项目结构** → **SDK** 选择对应版本。
